    🔧 API Reference#

    Journey Simulation API#

    Start Journey Simulation#

    POST /api/journey/simulate
Content-Type: application/json

{
  "persona": "karen",
  "journey": "insurance-policy",
  "options": {
    "thinkTime": 3000,
    "errorRate": 0.05,
    "businessContext": {
      "campaign": "summer-promo",
      "channel": "web-mobile"
    }
  }
}

    Get Journey Status#

    GET /api/journey/status/{journeyId}

Response:
{
  "journeyId": "abc123",
  "status": "completed",
  "persona": "karen",
  "duration": 4250,
  "steps": [
    {
      "name": "policy-discovery",
      "duration": 823,
      "status": "success"
    }
  ],
  "businessValue": 1200
}
